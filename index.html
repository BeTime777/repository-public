Politique de Confidentialité - BeTime

Dernière mise à jour : 29 septembre 2025

ID Qualia ("nous", "notre" ou "nos") exploite l'application mobile BeTime (ci-après désignée par le "Service").

Cette page vous informe de nos politiques en matière de collecte, d'utilisation et de divulgation des données personnelles lorsque vous utilisez notre Service.

Nous utilisons vos données uniquement pour fournir et améliorer le Service. En utilisant le Service, vous acceptez la collecte et l'utilisation des informations conformément à cette politique.

1. Données Collectées

1.1 Données personnelles

Nous collectons les données suivantes :

  • Adresse email : utilisée uniquement pour l'authentification et la gestion de votre compte

1.2 Données de jeu

  • Scores et progression dans le jeu
  • Statistiques de jeu
  • Préférences de l'utilisateur

1.3 Données collectées automatiquement

Lorsque vous utilisez notre Service, nous collectons automatiquement certaines informations via nos partenaires :

  • Données d'utilisation de l'application (Firebase Analytics)
  • Données publicitaires (AdMob et partenaires de médiation)
  • Identifiants techniques de l'appareil
  • Données de crash et de performance

2. Utilisation des Données

Nous utilisons vos données pour :

  • Créer et gérer votre compte utilisateur
  • Sauvegarder votre progression dans le jeu
  • Améliorer l'expérience utilisateur
  • Analyser l'utilisation de l'application
  • Afficher des publicités pertinentes
  • Assurer la sécurité et prévenir la fraude

Important : Nous ne vendons jamais vos données personnelles à des tiers.

3. Services Tiers Utilisés

Notre application utilise les services tiers suivants :

3.1 Firebase (Google)

  • Firebase Authentication : gestion de l'authentification
  • Firebase Database : stockage des données de jeu
  • Firebase Analytics : analyse d'utilisation
  • Politique de confidentialité : https://firebase.google.com/support/privacy

3.2 AdMob et Médiation Publicitaire

  • Google AdMob : affichage de publicités
  • Partenaires de médiation : réseaux publicitaires tiers
  • Ces services peuvent collecter des identifiants publicitaires et des données d'utilisation
  • Politique de confidentialité AdMob : https://support.google.com/admob/answer/6128543

Les partenaires publicitaires peuvent utiliser des cookies et des technologies similaires pour personnaliser les annonces. Vous pouvez gérer vos préférences publicitaires dans les paramètres de votre appareil iOS.

4. Stockage et Sécurité des Données

Vos données sont stockées de manière sécurisée via les serveurs Firebase (Google Cloud Platform) situés dans l'Union Européenne lorsque c'est possible.

Nous mettons en œuvre des mesures de sécurité techniques et organisationnelles appropriées pour protéger vos données contre tout accès, modification, divulgation ou destruction non autorisés.

5. Durée de Conservation

Nous conservons vos données personnelles aussi longtemps que votre compte est actif ou selon les besoins pour vous fournir le Service.

Après suppression de votre compte, vos données personnelles sont supprimées dans un délai maximum de 30 jours.

6. Vos Droits (RGPD)

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez des droits suivants :

  • Droit d'accès : obtenir une copie de vos données
  • Droit de rectification : corriger vos données inexactes
  • Droit à l'effacement : supprimer vos données
  • Droit à la limitation : limiter le traitement de vos données
  • Droit d'opposition : vous opposer au traitement de vos données
  • Droit à la portabilité : récupérer vos données dans un format structuré

6.1 Suppression de compte

Pour supprimer votre compte et toutes vos données associées, envoyez un email à : betime_support@gmail.com

Nous traiterons votre demande dans les 48 heures et supprimerons définitivement vos données dans les 30 jours.

7. Données des Mineurs

Notre Service n'est pas destiné aux personnes de moins de 13 ans. Nous ne collectons pas sciemment de données personnelles auprès d'enfants de moins de 13 ans. Si vous êtes parent ou tuteur et que vous savez que votre enfant nous a fourni des données personnelles, veuillez nous contacter.

8. Transferts Internationaux de Données

Vos données peuvent être transférées et traitées dans des pays autres que votre pays de résidence, notamment aux États-Unis où sont situés les serveurs de Google/Firebase.

Ces transferts sont effectués conformément au RGPD et aux clauses contractuelles types de l'UE.

9. Modifications de cette Politique

Nous pouvons mettre à jour notre Politique de Confidentialité périodiquement. Nous vous informerons de tout changement en publiant la nouvelle Politique de Confidentialité dans l'application.

La date de "dernière mise à jour" en haut de cette politique sera modifiée lors de toute révision.
